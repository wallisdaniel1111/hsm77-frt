# HSM-77 Fitness Readiness Tool (FRT)

**Vigila Et Percute**

## Mission Overview

The HSM-77 FRT is a mobile-friendly tool designed for Saberhawks training at **NAF Atsugi**.

## Local Specs

* **Atsugi Track:** Calibrated for 7 Laps (1.5 Miles).
* **Gym Logic:** Set Treadmill to 1.0% Incline.
* **Standard:** Navy Guide-5 Table 4-1 (Sea Level).

## Deployment

Hosted via Netlify. Data is stored locally on the user's device.
